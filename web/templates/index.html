🔍 欣亞 Sinya / 原價屋 CoolPC / Autobuy — 搜尋無結果

爬蟲使用關鍵字搜尋各商店的 API（如 Sinya JSON API、CoolPC WooCommerce API、Autobuy 類別頁面），若該產品在商店的商品目錄中未被列出，則無結果。

⚠️ 原價屋 CoolPC — 結果被廣告/開箱文淹沒

CoolPC 的 WooCommerce 將部落格文章、促銷活動也視為「商品」，爬蟲會過濾掉標題含【開箱、搶購、雙11、筆電、已截止】等關鍵字的項目。過濾後常常剩下 0 個可匹配的結果。

❌ 關鍵字匹配不足（≥2 個關鍵字 + 分數 ≥15 才匹配成功）

爬蟲從產品名稱中提取關鍵字（排除品牌通用詞如 nvidia、amd、intel、corsair 等），逐一比對候選商品標題。匹配規則為：

  • 至少 2 個關鍵字出現在標題中（否則跳過） — 例如搜尋 "990 Pro" 時，若商品標題只有 "990" 而沒有 "Pro"，則不匹配。
  • 變體懲罰 −60：若候選商品含有 GRE、XTX、Super、Ultra、Lite、Max、Ti 等後綴，但目標產品沒有，則嚴厲扣分（防止 RX 9070 GRE 匹配 RX 9070）。
  • 組合包懲罰 −50：標題含有搭機、套裝、主機、送、救贖、福利等字眼。
  • 完全比對加分 +100：產品名稱完整出現在標題中時，獲得額外加分。
  • 最低分數門檻 ≥15：2 個純關鍵字匹配 = 20 分，低於 15 分視為匹配失敗。

匹配失敗時，該商店對該產品使用模擬價格。
